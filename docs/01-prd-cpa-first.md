# CPA 1차 회계/세법 합격 운영체계 PRD

## 1. 제품 비전

CPA 1차 회계학과 세법개론을 준비하는 수험생에게 "무엇을 더 공부해야 하는가"보다 더 중요한 질문인 "지금 이 상태에서 합격 가능성을 가장 크게 올리는 다음 행동은 무엇인가"에 답한다.

이 제품은 기출문제 앱, 해설 앱, AI 챗봇이 아니다. 문제풀이 데이터, 합격수기, 합격자 인터뷰, 사용자 학습 로그를 합쳐 개인별 합격 전략을 생성하는 학습 운영체계다.

## 2. 첫 타깃

- 시험: CPA 1차
- 과목: 회계학, 세법개론
- 사용자: CPA 1차를 3개월 이상 준비할 의지가 있고, 현재 실력과 남은 기간에 맞는 과목별 우선순위를 알고 싶은 수험생
- 제외: 2차 답안 작성, 전 과목 종합 서비스, 강의 판매, 단순 커뮤니티

## 3. 해결할 문제

수험생은 대체로 다음 문제를 겪는다.

- 기출을 풀어도 어떤 개념이 약한지 정밀하게 모른다.
- 합격수기를 많이 읽어도 자기 상태에 맞는 전략으로 바꾸지 못한다.
- 회계학은 시간 압박, 계산량, 조건 누락 때문에 실력이 점수로 잘 전환되지 않는다.
- 세법개론은 개정, 말문제, 계산문제, 휘발성 암기가 섞여 우선순위가 무너진다.
- 공부량은 많은데 점수 상승에 기여하는 공부인지 검증하기 어렵다.

## 4. 핵심 가치 제안

사용자는 매일 다음을 받는다.

- 오늘 풀어야 할 문제 세트
- 반드시 복습해야 할 개념
- 버리거나 나중으로 미룰 유형
- 최근 오답의 원인 분석
- 합격수기와 인터뷰에서 검증된 행동 규칙
- 현재 합격 리스크와 다음 주 목표 점수

## 5. MVP 기능

### 5.1 진단

- 회계학/세법개론 기본 진단 테스트
- 과목별 정답률, 시간 초과율, 오답 원인 기록
- 개념별 숙련도 추정

### 5.2 문제 지능 데이터

각 문제는 아래 데이터로 구조화한다.

- 필요한 선수 개념
- 핵심 개념
- 풀이 시작점
- 계산 순서
- 함정
- 시험장 시간 전략
- 변형 가능 포인트
- 오답 유형
- 합격 점수 기여도

### 5.3 합격 전략 데이터

공개 합격수기와 인터뷰에서 다음 데이터를 추출한다.

- 수험 시작점
- 기간별 공부 전환점
- 과목별 회독 순서
- 객관식 진입 기준
- 기출과 모의고사 활용 방식
- 버림 전략
- 불합격 회피 규칙

### 5.4 개인별 처방

사용자 상태와 전략 데이터를 연결해 주간/일간 처방을 만든다.

- 오늘의 핵심 과목
- 오늘의 개념 3개
- 오늘의 문제 세트
- 오늘 버릴 것
- 이번 주 검증 지표

## 6. 비기능 요구사항

- 모든 AI 출력은 근거 데이터를 남겨야 한다.
- 공개 합격수기 원문을 서비스에 재배포하지 않는다.
- 저작권이 있는 문제 원문은 권리 확보 전까지 내부 분석/메타데이터 중심으로 다룬다.
- 세법 개정사항은 연도별 버전을 분리한다.
- "합격 보장" 표현은 사용하지 않는다. 합격확률은 리스크 지표로 표현한다.

## 7. 성공 지표

초기에는 매출보다 학습 성과 지표를 먼저 본다.

- 진단 후 4주 유지율
- 주간 계획 수행률
- 오답 원인 재발률 감소
- 회계학 시간 초과율 감소
- 세법 말문제/계산문제별 정답률 변화
- 사용자가 다음 행동을 결정하는 데 걸리는 시간 감소

## 8. 공식 기준

CPA 시험 시행계획과 과목 범위는 금융위원회 및 금융감독원 공인회계사시험 공고를 기준으로 한다.

- 2026년도 제61회 공인회계사시험 시행계획 공고: https://www.fsc.go.kr/po040200/85723
- 공인회계사시험 홈페이지: https://cpa.fss.or.kr

## 10. 첫 8주 실행안

1. CPA 1차 회계학/세법개론 기출 메타데이터 구조 설계
2. 공개 합격수기 200개 수집 목록화
3. 문제 100개를 수작업으로 지능 태깅
4. 합격자 인터뷰 10명 진행
5. 전략 규칙 50개 생성
6. 진단 테스트와 주간 처방 프로토타입 출시
